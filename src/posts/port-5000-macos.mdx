title: Port 5000 busy by macOS control centre
tags: mac, macos, networking
summary: Port 5000 busy? No problem.
date: 2023-03-12
---

By running `portPid 5000` (aliased to `lsof -i :5000`), I found that the process that was using that port was macOS Control Centre.

This is due to the option `airplay receiver` checked in the macOS Settings
